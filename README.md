# AI Grocery Planner

A standalone grocery planning web app hosted at [groceries.streamfuse.app](https://groceries.streamfuse.app).

Source lives in the StreamFuse monorepo under `ai_grocery_planner/` for deployment via Firebase Hosting. Public mirror: [github.com/NamelessNATM/AIGroceryPlanner](https://github.com/NamelessNATM/AIGroceryPlanner).

This directory does not include Firebase server credentials or StreamFuse app keys. See [STANDALONE_REPO.md](STANDALONE_REPO.md) for syncing this folder to the public GitHub repository.
